## 🧠 Overview

**ASUS Armoury Crate** is an all-in-one utility for configuring and monitoring compatible ASUS and ROG devices.

It offers centralized control over RGB lighting, system performance, cooling, and hardware settings through a modern dashboard.

Built-in maintenance tools make it easy to manage updates, device profiles, and performance modes.

Designed for gamers, PC enthusiasts, creators, and Windows users with ASUS hardware.

## 🚀 Key Features

| Feature | Description |
|----------|-------------|
| 🌈 **Aura Sync Lighting** | Customize and synchronize RGB effects across compatible ASUS devices. |
| ⚡ **Performance Modes** | Instantly switch between silent, balanced, performance, and turbo profiles. |
| 🌡️ **Cooling Management** | Configure fan behavior and optimize thermal performance. |
| 📈 **Hardware Monitoring** | Track CPU, GPU, memory, temperatures, and system activity in real time. |
| 🎮 **Application Profiles** | Automatically apply customized settings for games and software. |
| 🔄 **Update Center** | Download drivers, firmware, and utility updates directly from the app. |
| 🖥️ **Device Configuration** | Manage compatible motherboards, laptops, graphics cards, and peripherals. |
